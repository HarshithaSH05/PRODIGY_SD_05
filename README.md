##  Web Scraping

This project is an advanced web scraping application developed using Python and Streamlit.

The application extracts product information from an online e-commerce website and stores the data in a structured CSV format.

### Extracted Information

- Product Name
- Price
- Rating
- Page Number

### Features

✔ Multi-page web scraping  
✔ Extracts 100+ products  
✔ Interactive Streamlit interface  
✔ Progress bar indicator  
✔ Data table visualization  
✔ CSV file download  

### Technologies Used

- Python
- Streamlit
- BeautifulSoup
- Requests
- Pandas

### Installation

Install dependencies:

pip install -r requirements.txt

### Run Application

streamlit run app.py
